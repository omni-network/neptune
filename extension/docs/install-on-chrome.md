# Install on Chrome


1. Build the extension.

  ```
  pnpm build

  # or

  yarn build

  # or

  npm run build
  ```

2. Go to `chrome://extensions`
3. Turn on developer mode (top right)
4. Click "Load uppacked"
5. Select the build directory from the build step (`<neptune_root>/extension/dist/`)
6. Have fun forking!
